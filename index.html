The chart above shows the number of households and the percentage of pet households for each state in the United States from 2012. It gives an introduction to how many households are in the U.S. and how many of those households actually own pets.

I liked that this chart shows the same information, but features the largest state based off of the number of households and pet household percentage. I also liked that the chart allows the user to immediately see the states with the largest amount of pet households.

However, the map does not label all of the circles, which indicates the states included in the data. The user has to physically hover over the smaller circles to show the name of the state with the information. The circle chart makes it a little difficult to see which state has the lowest percentage of pet households or lowest number of households in the U.S.

I prefer the map to the circle graph, because I can obtain more information than what is displayed. The map also allows me to analyze the data further. I can locate regional trends for pet households, compare pet households by region and the map gives me a better visual of the locations.
